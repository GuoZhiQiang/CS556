📐 行列式的 10 条重要性质

用 3×3 方阵直观解释 · 从几何到代数

🎯 行列式是什么？ 一个将方阵映射为实数的函数，表示向量张成的“有向体积”。
下面每一条性质都结合具体的 3×3 矩阵示例说明，帮助你建立直觉。
1 单位矩阵的行列式为 1

单位矩阵 \(I_3\) 是 3×3 方阵，对角线上为 1，其余为 0。

I₃ = ⎡1 0 0⎤ ⎢0 1 0⎥ ⎣0 0 1⎦
det(I₃) = 1 × 1 × 1 = 1

几何意义：单位立方体的有向体积为 1。

2 交换两行改变符号

交换矩阵的任意两行，行列式变号。

A = ⎡1 0 0⎤ 交换第1行和第3行 → B = ⎡0 0 1⎤ ⎢0 1 0⎥ ⎢0 1 0⎥ ⎣0 0 1⎦ ⎣1 0 0⎦
det(A) = 1, det(B) = -1

✍️ 几何原因：交换向量顺序 → 定向反转（右手系变左手系）。

3 对每一行具有线性性质

行列式关于矩阵的每一行都是线性的，包含两个方面的含义：

  • 数乘：若某一行所有元素乘以常数 \(k\)，则行列式也乘以 \(k\)。
  • 可加性：若某一行可写成两个行向量之和，则行列式等于两个行列式之和（其余行不变）。
例1（数乘）：A = ⎡1 2 3⎤ 将第一行乘以2 → B = ⎡2 4 6⎤ ⎢4 5 6⎥ ⎢4 5 6⎥ ⎣7 8 9⎦ ⎣7 8 9⎦ det(B) = 2·det(A)
例2（可加性）：考虑 C = ⎡1+4 2+5 3+6⎤ = ⎡5 7 9⎤ ⎢4 5 6 ⎥ ⎢4 5 6⎥ ⎣7 8 9 ⎦ ⎣7 8 9⎦ 则 det(C) = det⎛⎡1 2 3⎤⎞ + det⎛⎡4 5 6⎤⎞ ⎝⎢4 5 6⎥⎠ ⎝⎢4 5 6⎥⎠ ⎣7 8 9⎦ ⎣7 8 9⎦ 右边第二个行列式因为两行相同而为0，左边第一个也为0，故det(C)=0。

💡 线性性质是行列式公理化的核心之一，也是拉普拉斯展开的理论基础。

4 若两行相同，则行列式为 0

矩阵中有两行完全相同，则行列式为零。

A = ⎡1 2 3⎤ ⎢1 2 3⎥ (第1行 = 第2行) ⎣4 5 6⎦
计算：按第1行展开，或观察两行成比例 → det(A) = 0

几何解释：两个向量相同 → 平行六面体坍缩 → 体积为零。

5 将一行的倍数加到另一行上，行列式不变

这是高斯消元的基础，不改变行列式的值。

A = ⎡1 0 0⎤ 将第1行的2倍加到第2行 → B = ⎡1 0 0⎤ ⎢0 1 0⎥ ⎢2 1 0⎥ ⎣0 0 1⎦ ⎣0 0 1⎦
det(A) = 1, det(B) = 1 × (1×1 - 0×0) = 1

💡 体积不变，因为剪切变换不改变平行六面体的底面积和高。

6 某行全为零 ⇒ 行列式为 0

如果矩阵有一行全部是零，则行列式为零。

A = ⎡1 2 3⎤ ⎢0 0 0⎥ (第2行全零) ⎣4 5 6⎦
按第二行展开，每一项都包含0因子 → det(A) = 0

几何意义：一个向量为零 → 张成的体积为零。

7 三角矩阵的行列式等于对角线元素乘积

上三角或下三角矩阵，行列式即主对角线上元素的乘积。

A = ⎡2 1 3⎤ (上三角) ⎢0 4 5⎥ ⎣0 0 6⎦
det(A) = 2 × 4 × 6 = 48

下三角同理。可直接从展开公式看出，非对角项在展开中消失。

8 奇异矩阵 ⇔ det(A)=0；可逆 ⇔ det(A)≠0

矩阵不可逆（奇异）当且仅当行列式为0，反之可逆当且仅当行列式非零。

奇异矩阵例：A = ⎡1 2 3⎤ (第2行 = 2×第1行) ⎢2 4 6⎥ ⎣7 8 9⎦ det(A) = 0
可逆矩阵例：B = ⎡1 0 0⎤ ⎢0 2 0⎥ ⎣0 0 3⎦ det(B) = 6 ≠ 0

🌍 行列式为零 ↔ 体积零 ↔ 列向量线性相关 ↔ 矩阵奇异。

9 det(AB) = det(A) · det(B)

矩阵乘积的行列式等于各自行列式的乘积。

A = ⎡1 0 0⎤, det=2 B = ⎡0 1 0⎤, det=-1 ⎢0 2 0⎥ ⎢1 0 0⎥ ⎣0 0 1⎦ ⎣0 0 1⎦
AB = ⎡0 1 0⎤, det(AB) = -2 = 2 × (-1) ⎢2 0 0⎥ ⎣0 0 1⎦

几何含义：线性变换的体积缩放因子相乘。

10 det(Aᵀ) = det(A)

矩阵转置不改变行列式的值。

A = ⎡1 2 3⎤ Aᵀ = ⎡1 4 7⎤ ⎢4 5 6⎥ ⎢2 5 8⎥ ⎣7 8 9⎦ ⎣3 6 9⎦ det(A) = 0 ⇒ det(Aᵀ) = 0
对可逆例：A = ⎡1 0 1⎤, det=4；转置后行列式仍为4。 ⎢0 2 0⎥ ⎣1 0 3⎦

因为行和列在行列式定义中地位对称。

📌 一句话总结

行列式是 有向体积 的函数：正负表示定向，绝对值表示体积大小。这10条性质共同刻画了体积在初等行变换下的行为，也是线性代数中判断可逆性、计算特征值乘积的核心工具。
